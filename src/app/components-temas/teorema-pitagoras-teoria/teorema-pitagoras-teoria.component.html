Teorema de Pitágoras

Alexander, D. C., & Koeberlein, G. M. (2015). Elementary Geometry for College

Students (6th ed.). Cengage Learning.

El Teorema de Pitágoras es uno de los resultados más importantes en geometría euclidiana, fundamental para entender las relaciones entre los lados de un triángulo rectángulo.

En un triángulo rectángulo, que tiene un ángulo recto (90 grados), el teorema establece que: c2 = a2 + b2 = c = √a2 + b2

Donde:

• c es la longitud de la hipotenusa, el lado opuesto al ángulo recto.

• a y b son las longitudes de los catetos, los lados que forman el ángulo recto. Este teorema puede ser utilizado para encontrar la longitud de cualquiera de los lados de un triángulo rectángulo si se conocen los otros dos.

Para calcular la hipotenusa c de un triángulo rectángulo cuando se conocen las longitudes de ambos catetos a y b, el procedimiento es el siguiente:

• Identifica los valores dados: a y b son las longitudes de los catetos conocidos.

• Aplica el Teorema de Pitágoras: Utiliza la fórmula del teorema para calcular c = √a2 + b2

• Calcula el valor numérico: Sustituye los valores conocidos de a y b en la fórmula y realiza las operaciones necesarias para obtener c.

Para encontrar la longitud de uno de los catetos a o b en un triángulo rectángulo cuando se conoce la hipotenusa c y otro cateto, el procedimiento es:

Identifica los valores dados

• c es la longitud de la hipotenusa.

• hipotenusa es la longitud de un cateto conocido.

Aplica el Teorema de Pitágoras:

• Despeja a o b en la fórmula del teorema: a2 = c2 − b2 o b2 = c2 − a2

• Luego, toma la raíz cuadrada para obtener a o b: a = √c2 − b2 o b = √c2 − a2

Calcula el valor numérico: Sustituye los valores conocidos de c y el otro cateto en la fórmula y realiza las operaciones necesarias para obtener a o b.
